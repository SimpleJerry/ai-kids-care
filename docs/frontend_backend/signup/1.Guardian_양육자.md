updated_at : 현재시간


frontend, backend 매칭해서 작업하기

4-1) 관계 Display 구체화
[디폴트 선택] 디폴트 선택은 `- 선택 -` 나오게하고, 반드시 관계 선택해야 회원가입되게 체크 부탁해

구체화 부분은 성별이 변경된 경우 관계 공통 코드의 parent_code를 읽어서 성별에 맞게 관계 데이터가 일관되게 하는 것이다.
ex) 위의 성별이 여자의 경우 엄마, 외할머니, 친할머니, 기타가 나오게 한다.
