## frontend
input box 대상 : 이름, 로그인ID ,이메일, 전화번호, 비밀번호, 비밀번호 확인, 행정청 부서명

## Backend

1.Database (postgresql)

insert table 3 대상

1.1 users 테이블에 insert
user_id : 자동생성
login_id : 로그인 ID
email: 이메일
phone : 전화번호
password_hash : 비밀번호 -> jwt 를 활용하여 암호화 시켜서 저장할수 있게 함.
status : ACTIVE
last_login_at : 현재시간
created_at : 현재시간
updated_at : 현재시간

1.2 superadmin 테이블에 insert
  superadmin_id : 자동생성성
  user_id : user_account 테이블의 user_id 동일
  name : 이름
  department : 화면의 행정청 부서명 input box 값
  created_at : 현재시간
  updated_at : 현재시간

1.3 user_role_assignment insert
  role_assignment_id : increment
  user_id : USER_ACCOUNT.user_id
  role user_role_enum : SUPERADMIN
  scope_type : PLATFORM
  scope_id : null
  status status_enum : ACTIVE
  granted_at : 현재시간
  granted_by_user_id : users.user_id
  revoked_at : null
