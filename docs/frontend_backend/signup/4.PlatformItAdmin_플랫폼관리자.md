## Backend

1.Database (postgresql)

insert table 2 대상

1.1 users  테이블에 insert

user_id : 자동생성
login_id : 로그인 ID
email: 이메일
phone : 전화번호
password_hash : 비밀번호 -> jwt 를 활용하여 암호화 시켜서 저장할수 있게 함.
status : ACTIVE
last_login_at : 현재시간
created_at : 현재시간
updated_at : 현재시간

1.2 user_role_assignment insert
  role_assignment_id : increment
  user_id : users.user_id
  role user_role_enum : PLATFORM_IT_ADMIN
  scope_type : PLATFORM
  scope_id : null
  status status_enum : ACTIVE
  granted_at : 현재시간
  granted_by_user_id : users.user_id
  revoked_at : null
