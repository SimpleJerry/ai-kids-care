## instruction-kidsPopupForSignup.md

# 1. 회원가입

[전제조건] 각 권한별 회원가입의 테이블 insert 대상이 다름

http://localhost/signup


### 회원유형 양육자 선택된 상태에서 회원가입 버튼을 클릭하는 경우 frontend,backend 작업해줘

frontend의 화면 비밀번호 밑에 kids 찾기 팝업 추가해야함.
아이 이름을 입력하고 엔터를 치면 아이 찾기 팝업이 호출됨.
아이 이름 입력박스 옆에 아이 찾기 버튼 추가해서 아이 찾기 팝업 호출.

## 아이 찾기 팝업
아이 이름 상단에 검색해서 조회

조회 테이블 : child
이름으로 조회하고 기본 정보는 테이블 형태로로 Display 함.

child_guardian_relationship 에 insert 할 child_id 컬럼의 값을 hidden 데이터로 화면에 가지고 있어야 함. 

그래서 팝업이 닫히면 Signup 화면에 child_id를 셋업해 주어야 함.
